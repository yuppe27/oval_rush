# OVAL RUSH

90年代アーケードレーシングの爽快感をブラウザ上で再現した3Dレーシングゲームです。
制限時間をチェックポイント通過で延長しながら、ライバル車を抜き去って上位入賞を目指しましょう。

---

## インストール・起動方法

### 必要環境

- モダンブラウザ（Chrome 100+ / Firefox 100+ / Edge 100+ / Safari 16+）
- Python 3 または Node.js（ローカルサーバー用）

### 起動手順

**Windows の場合（推奨）:**

```bat
start_game_server.bat をダブルクリック
```

**Linux / macOS の場合（推奨）:**

```bash
chmod +x ./start_game_server.sh
./start_game_server.sh
```

**手動起動（Python）:**

```bash
python3 -m http.server 8000
```

`python3` が無い環境では `python -m http.server 8000` でも起動できます。

**手動起動（Node.js）:**

```bash
npx serve .
```

起動後、ブラウザで `http://localhost:8000` を開いてください。

> **注意:** `file://` プロトコルでは ES Modules の制限によりゲームが起動しません。必ずローカルサーバー経由でアクセスしてください。

## 遊び方

### ゲームフロー

```
タイトル画面
  → モード選択（Arcade Mode / Time Attack / Free Run）
  → コース選択
  → 車両選択
  → 難易度選択
  → トランスミッション選択（AT / MT）
  → レース開始
  → リザルト画面
```

### 基本ルール

- レース開始時に**制限時間**が与えられます
- コース上の**チェックポイント**を通過するたびに制限時間が延長されます
- 制限時間が尽きると **GAME OVER**
- 制限時間内にゴールすれば**順位が確定**し、ランキングに登録できます

| コース | 周回数 | 初期時間 | CP延長 |
|--------|--------|----------|--------|
| Thunder Oval Speedway | 8周 | 40秒 | +12秒 |
| Seaside Grand Circuit | 4周 | 45秒 | +14秒 |
| Mountain Apex Rally | 2周 | 50秒 | +16秒 |

## アクセシビリティ（3D酔い緩和モード）

レース中に `V` キーを押すと**コンフォートフレーム**の表示を切り替えられます。
画面には常に動かない参照フレーム（コックピット風の上下バー・水平基準線・中央レティクル・
周辺ビネット）が重ねて描画され、カメラが揺れても視界の中に静止した基準点が残ります。

3D酔い・VR酔いは、身体が感じている動き（前庭感覚）と目に入る動き（視覚）の不一致で
発生しやすいことが知られています。コンフォートフレームはその不一致を抑える補助機能です。

| 要素 | 役割 |
|------|------|
| 上部シェード | 視野の上端を固定し、画面上方の揺れを隠す |
| 下部ダッシュボード風バー | コックピット感を再現し、画面下方の基準を安定化 |
| 中央水平線 | 画面中央に固定された水平基準 |
| 中央レティクル | 注視点の固定によりサッケード（目の跳躍運動）を抑制 |
| 周辺ビネット | 周辺視野の動き情報を緩やかに減衰（トンネル効果） |

設定はブラウザの `localStorage` に保存されるため、次回以降も同じ状態で起動します。
併せて、以下の操作も酔いの軽減に有効です。

- カメラを**バンパー視点**または**俯瞰視点**に切り替える（`C` キー）
- 難易度を **EASY** にして速度・加減速を抑える
- 画面との距離を取り、適度に休憩を挟む

## 車両紹介

### Falcon MK-I（赤）— バランス型

- 最高速: **286 km/h**
- すべての性能が均等で扱いやすい万能型
- 初めて遊ぶ方に最もおすすめ
- **推奨コース:** 全コース

### Bolt RS（黄）— スピード型

- 最高速: **326 km/h**（3車種中最速）
- 直線では圧倒的だが、コーナーでの扱いは難しい
- ドリフト時の姿勢変化が大きいため上級者向け
- **推奨コース:** Thunder Oval（長い直線を活かせる）

### Ironclad GT（オレンジ）— グリップ型

- 最高速: **270 km/h**
- 加速力が最も高く、コーナーが安定している
- 最高速は低いが、テクニカル区間で特に強い
- **推奨コース:** Mountain Apex（急カーブ連続区間で真価発揮）

## コース紹介

### Thunder Oval Speedway ★☆☆（初級）

バンク角30°の高速コーナーと長い直線が特徴のオーバル型コース。
コース幅が広く（20m）、横並びでのバトルが楽しめます。
背景には巨大スタジアムや飛行船が登場します。

### Seaside Grand Circuit ★★☆（中級）

S字カーブ、**トンネル区間**、**石畳区間（グリップ低下）**が含まれる本格サーキット。
チェックポイントが4箇所あり、効率よく通過することが重要です。
背景は紺碧の海岸線と石造りの街並みです。

### Mountain Apex Rally ★★★（上級）

標高差200mを駆け抜ける山岳コース。コース幅が狭く（13m）、急カーブが連続します。
**ジャンプスポット**が2箇所あり、**霧のゾーン**では視界が制限されます。
背景には深い森林、渓谷橋、古城、雲海が広がります。

## ドリフトシステム

### ドリフトの発動方法

以下の3条件を同時に満たすと**自動でドリフトが発動**します。

1. 最高速の **30% 以上**の速度が出ている
2. ブレーキを踏んだ後 **0.2秒以内**にステアリングを入れる
3. そのときのステアリング入力量が **78% 以上**

### ドリフトのメリット

- ドリフト終了時、**品質に応じたブーストが自動発動**（最大1.0秒）
- ブースト中は最高速の **120%** まで加速
- ドリフトを上手くコントロールするほどコーナー後の脱出速度が上がる

### ブーストの強さ

ブースト時間は基本値にスケール係数（0.45〜0.80×）が掛かります。
スケール係数はドリフト終了時の姿勢（アライメント）とスロットル入力で決まります。

| ドリフト評価 | 基本値 | 実効範囲 |
|------------|--------|---------|
| 低 | 0.22 秒 | 約 0.10〜0.18 秒 |
| 中 | 0.55 秒 | 約 0.25〜0.44 秒 |
| 高 | 1.0 秒 | 約 0.45〜0.80 秒 |

## スリップストリーム

前方の車両の後方 **3〜15m** 以内、かつ進行方向から **±10°** 以内の位置を走ると、
スリップストリームが発動して最高速の **103%** まで加速します。
HUD に「SLIPSTREAM XX%」と表示されたら積極的に活用しましょう。

---

## 難易度について

| 難易度 | AI速度 | ラバーバンド | タイム余裕度 |
|--------|--------|------------|-------------|
| EASY | 低め | 強（差を縮めやすい） | 高（係数1.35） |
| NORMAL | 標準 | 中 | 標準（係数1.18） |
| HARD | 高め | 弱（差がつきやすい） | 低（係数1.05） |

**ラバーバンド:** プレイヤーが大きくリードするとAIが加速し、遅れるとAIが減速する仕組みです。

## ヒント

- **チェックポイントは最優先で通過する。** 制限時間が切れると即GAME OVERなので、順位より時間管理を優先しましょう。
- **ドリフト後のブーストを連続させる。** コーナーごとにドリフト→ブーストを決めると大幅なタイム短縮になります。
- **スリップストリームを攻略に使う。** 前の車に近づいてスリップに入り、直線の終わりで一気に追い越しましょう。
- **ドリフト角度を保ちすぎない。** ドリフト角が **83°** を超えるとスピンします。カウンターステアで適度な角度（15°〜40°）を維持するのが理想です。
- **Mountain Apexのジャンプスポットでブーストを温存する。** ジャンプ直前のドリフトを成功させると、着地後にブーストが乗って有利になります。
- **Bolt RSはThunder Ovalで真価を発揮する。** 長い直線が続くオーバルコースなら、最高速326km/hのスピードを最大限に活かせます。
- **HARD難易度ではタイム余裕度が低い。** 初期時間・CP延長ともに短く計算されるため、チェックポイントのロスを最小限にするラインどりが重要です。
- **MTを使いこなすとコーナー立ち上がりが速くなる。** ドリフト中に低いギアを維持し、出口で素早くシフトアップすると加速が鋭くなります。
